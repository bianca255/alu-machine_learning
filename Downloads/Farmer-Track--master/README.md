# FarmTrack (scaffold)

This repo is a **Flutter project layout only** (empty folders + `.gitkeep`). Teammates add Dart and platform files under `lib/`, `android/`, `ios/`, etc.

The complete app (if you need to copy from it) lives on branch **`full-source`**:

```bash
git fetch origin
git checkout full-source -- lib android ios
# or: git switch full-source
```

```bash
flutter pub get
flutter run
```
