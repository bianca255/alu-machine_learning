# Bayesian Probability

This directory contains implementations of Bayesian probability functions using the binomial distribution, applied to a study on a revolutionary cancer drug.

## Files

- `0-likelihood.py` - Calculates the likelihood of obtaining data given various hypothetical probabilities
- `1-intersection.py` - Calculates the intersection of obtaining data with various hypothetical probabilities
- `2-marginal.py` - Calculates the marginal probability of obtaining the data
- `3-posterior.py` - Calculates the posterior probability for various hypothetical probabilities given the data
